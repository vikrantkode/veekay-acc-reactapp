
# Ecommerce App

Veekay Accessories is an e-commerce app made with ReactJS and JWT



## Tech Stack

**Client:** `ReactJS`, `CSS`, `JWT`

**Hosting Platform:** `Netlify`


## Features

- Home Page
- Product Listing Page
- Add To Cart
- Add TO Wishlist
- Remove from Cart
- Increase/Decrease Quantity In Cart
- Filters
- Login/SignUp Page
